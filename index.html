Projects

tāstr is a place to find and share recipes. A user can sign up, log in, upload photos, and like recipes posted by other users.
tāstr is a full CRUD app built with Ruby on Rails, styled with Bootstrap, sprinkled with some JavaScript and jQuery, and hosted on Heroku (patience is a virtue).

try a taste

Dots & Boxes is a two-player game in which players attempt to complete boxes by selecting the lines that connect dots. When a player completes a box, the box takes the color of the player who completed it. The game is over when all of the boxes are completed, and the player with the most boxes wins.
I made Dots & Boxes using HTML, CSS, and JavaScript. I utilized jQuery for simpler DOM manipulation.
